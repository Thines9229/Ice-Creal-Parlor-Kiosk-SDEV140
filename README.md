# Ice-Creal-Parlor-Kiosk-SDEV140

# This application is meant to modernize and streamline the operations of a dessert retail business, turning manual error broadcast into intuitive digital experience. It acts as an interactive kiosk allowing users to navigate select and purchase products through visual elements rather than commands

# Reasons

#	To enhance customer experience by reducing waiting time
#	To allow people to take their time browsing for their favorite flavor
#	To minimize or completely get rid of miscommunication between customer and staff
# Having a kiosk register that the people can control themselves, can save time for the workers and minimize labor costs so that the workers only have to deal with the orders

# Goals and Accomplishments

# I want the application to provide the user with options of their favorite flavors, allowing them to choose how many scoops they would like and at the end tell them about their total cost for each scoop.

# Target audience

# Anyone who wants a sweet treat or is feeling overwhelmed on a hot day and needs a little refresher
# Age: 5+
# Female & Male
# From any background and any income or education level.
# For any job, it can be seen as a treat after a long day at work!!


# Step 1: Before you run the program
# Before you run the program make sure you have everything in the right folder (they should all be in the same folder)

# Step 2: Run program
# Now it’s time to run the program, after clicking the run button 

# When the program successfully starts, the window will first look like this, NO NOT CLICK EXIT. I know it looks tempting but that’s not the game plan. To see the full program, click the square on the top

# Step 3: Have fun with the application
# Now that you have full access to the application. The next thing to do is do what the application is intended for……BUYING YOU FAVORITE ICE CREAM SCOOPS!!!! (YAY)

# What you want to do now is see what ice cream flavor out of my small selection…. (☹) that you would like and use the spin box to choose the scoop amount you would like! You can get 3 scoops for each flavor, but you don’t want a stomachache!!

# After you are satisfied with your ice cream haul, to get the total price amount that you owe there should be a order button on the lower middle part of the window.

# When you click the button, you should get a total price of ALLLLL the ice cream you got.
# After you get your total, and you “paid” then you may close the window by clicking the ”Close” button.

# Step 4: Close the program
# Now that everything is done, you can go on your merry way with your ice cream, just don’t forget to click the “exit” button on the top of the screen to mark order completed.
